oncotools
=============================

A Python library for interaction with the Oncospace Database,
image transformations, calculation of spatial dose features, and machine learning analysis.

Goals
-----

This package makes it possible to:

- connect to and query the Oncospace SQL database
- write simple scripts to easily interact with the data
- analyze the integrity of the data
- perform various types of analysis, including dosimetric and radio-morphologic analysis
